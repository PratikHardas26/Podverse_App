# Podverse - Podcast Streaming App

Podverse is a Flutter-based podcast streaming application integrated with Firebase services.

## 🚀 Features
- User Authentication (Firebase Auth)
- Upload Podcasts
- Podcast Categorization
- Admin Approval/Rejection System
- Cloud Firestore Database
- Firebase Storage for Media

## 🛠 Tech Stack
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## 📱 Modules
1. User Module
2. Admin Module

## 📦 Installation
flutter pub get
flutter run
